unreal-mcp-ue4

UE4.27.2-focused MCP server for Unreal Engine using Unreal Python Remote Execution.

Repository Portfolio About

Why it exists

Many MCP and Unreal automation examples assume UE5-era editor capabilities, but real projects often still live on UE4.27.2. This project focuses on practical workflows that remain realistic and useful in a UE4.27 production environment.

What it does

  • Reads project, map, asset, and actor information from an open Unreal Editor session
  • Exposes editor and content workflows through an MCP-friendly tool surface
  • Supports actors, assets, widgets, Blueprints, and inspection flows
  • Keeps the implementation grounded in what stock UE4.27 Python can support reliably
